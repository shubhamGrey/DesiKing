# Lottie Animation Files

Download free spice-themed animations from https://lottiefiles.com and save them here.

Required files (search terms listed):
- `chili.json` — search "chili pepper" or "spice"
- `leaf.json` — search "plant leaf" or "herb"
- `fire.json` — search "fire flame"
- `fork.json` — search "food restaurant"
- `star.json` — search "star sparkle" (for hero corner)

All animations are used with MUI icon fallbacks, so missing files degrade gracefully.
